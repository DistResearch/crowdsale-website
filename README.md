# Crowdsale Website

## How to run?

- Make sure Parity is running on default settings.

- Start the backend service:
    ```
    $ cd backend
    $ npm i
    $ npm start
    ```

- Start the frontend service:
    ```
    $ cd frontend
    $ npm i
    $ npm start
    ```

- Open [http://localhost:8080](http://localhost:8080) in the browser.
